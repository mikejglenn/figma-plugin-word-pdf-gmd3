# Figma Plugin (GMD3) - Word to PDF

On a Figma design go to menu -> Plugins -> Development -> Import plugin from manifest...

Select the manifest.json file. The plugin is now imported.

Then run the Figma Word to PDF (GMD3) plugin.

Go to menu -> Plugins -> Development -> Figma Word to PDF (GMD3)
